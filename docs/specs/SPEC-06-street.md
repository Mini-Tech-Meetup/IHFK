# SPEC-06: Street Stage

## 목표
한 화면 전체에서 떨어지는 키오스크 50대를 파괴한다.

## 사용자 동작
화면 어디든 계속 떨어지는 키오스크를 50대 부수고, 낙하가 멈추면 남은 키오스크를 더 부수거나 오른쪽 공장 출구로 간다.

## 기술 설계
Street가 약 2초 주기 이벤트, 7번째 이벤트 묶음, 활성 12대 cap, stage별 카운트를 소유한다.

## 의존 SPEC
SPEC-04, SPEC-09

## 제외 범위
전방 스폰, 공격하는 적, 무한 스크롤
