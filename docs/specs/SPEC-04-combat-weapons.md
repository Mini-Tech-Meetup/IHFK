# SPEC-04: Combat and Weapons

## 목표
빠른 연타, 키오스크 파손과 세 소모 무기를 구현한다.

## 사용자 동작
1~4로 즉시 무기를 바꾸고 Space를 홀드해 매우 빠르게 사용한다. 소진된 무기는 깨지며 주먹으로 자동 복귀한다.

## 기술 설계
Player가 무기별 cooldown·damage·range를 적용하고 GameSession이 잔량·상한·선택을 관리한다. Kiosk는 사망 가드로 중복 카운트를 차단한다.

## 의존 SPEC
SPEC-02, SPEC-03

## 제외 범위
인벤토리 화면, 재장전, 키오스크 간 연쇄 피해

## 구현 체크리스트
- [x] 주먹 10/s
- [x] 방망이 5~6/s, 10/20회
- [x] 전기톱 20tick/s, 4/8초
- [x] 샷건 4/s, 6/12발
- [x] 15% 동등 랜덤 드롭
- [x] 소진 시 자동 주먹
- [x] 빈 슬롯 선택 차단과 소진 순간 무기 파편·효과음
- [x] 파손 3단계·날아감·파편
- [x] 파괴 순간 30~60ms hit-stop·흰 flash·강한 shake

## 인수 조건
- [x] 일반 키오스크 주먹 약 1초 파괴
- [x] 연쇄 피해 없음
- [x] 파괴 수 중복 증가 없음

## 테스트
- [x] 자동: 잔량 추가·상한·소모·주먹 복귀
- [x] 데스크톱: 1~4 선택과 Space 홀드 E2E
- [x] 모바일: 강제 터치 무기 선택·공격 탭
- [x] 시각 검수: 5단계 래스터 파손·파편·세 무기 전신 애니메이션

## 증거
- 명령 및 결과: 브라우저 29/29, 래스터 무기 드롭 3종 실화면 검수, 최종 캠페인 E2E 통과
- 스크린샷: HUD 무기 4칸 및 전투 화면
- 성능 수치: fist 100ms, bat 180ms, chainsaw 50ms, shotgun 250ms
- 알려진 제한: 실제 터치 홀드 감각은 모바일 실기기 대기
- 상세: [polish validation](../evidence/2026-07-13-polish-validation.md)
