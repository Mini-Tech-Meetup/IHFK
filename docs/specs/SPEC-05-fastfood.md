# SPEC-05: Fast-food Stage

## 목표
맨주먹 조작을 익히는 첫 고정 화면을 구현한다.

## 사용자 동작
컷신에서 깨진 첫 키오스크를 지나 남은 한 대를 맨주먹으로 부수고 열린 오른쪽 문으로 나간다.

## 기술 설계
FastFood는 2대 중 1대를 이미 카운트한 세션 상태로 시작하고 두 번째 파괴 이벤트가 출구 collider를 해제한다.

## 의존 SPEC
SPEC-03, SPEC-04, SPEC-09

## 제외 범위
무기 드롭, 낙하 키오스크, 스크롤 카메라
